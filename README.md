# 博客系统demo——前后端分离项目

## 技术栈

前端：
Vue、Element UI

后端：
SpringBoot、MyBatis、MySQL

## 说明

此项目最初是我在学完vue后，写的练手项目，最初是年初写的。最初前端是用的死数据，后端也是为了配合前端写的接口。

由于只是demo，用了尽可能少的技术来实现博客的基本功能，但是后端写完后的接口一直没和前端对接，然后就鸽了。

最近又想起来此项目，把前端的死数据去掉后，数据与后端对接。把之前没写完的功能简单的补充了下，就开源出来了
